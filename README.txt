== Description ==

Create an attractive online help knowledge base from your WordPress dashboard with Blue Rooster Knowledge Base. This ready to use plugin has been designed and used by technical writers across the globe.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `content-manager.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to be BR Setting page and update "What would you like to call your knowledge base?".

This section describes how to create FAQs through the plugin.

1. Go to FAQs and click on Add New FAQs
2. Give it a title and add the post content.
3. Publish it and access it via /{blue-rooster-name}/faq
